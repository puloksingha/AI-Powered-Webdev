## 📅 Day 01 — HTML Fundamentals

### What is HTML?

**HTML (HyperText Markup Language)** is the backbone of every web page. It defines the structure and content of a webpage using a system of elements and tags.

### Topics Covered

- ✅ HTML document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- ✅ Headings (`<h1>` – `<h6>`)
- ✅ Paragraphs and text formatting (`<p>`, `<strong>`, `<em>`)
- ✅ Links and navigation (`<a href="">`)
- ✅ Images (`<img src="" alt="">`)
- ✅ Lists — ordered and unordered (`<ol>`, `<ul>`, `<li>`)
- ✅ Tables (`<table>`, `<tr>`, `<th>`, `<td>`)
- ✅ Forms and input elements (`<form>`, `<input>`, `<button>`)
- ✅ Semantic HTML elements (`<header>`, `<nav>`, `<main>`, `<footer>`)
- ✅ HTML comments

### Key Concepts

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to AI-Powered Web Development.</p>
  </body>
</html>
```

### AI Tip of the Day 💡

> Use **GitHub Copilot** or **ChatGPT** to generate boilerplate HTML structures instantly. Simply describe what you want — _"Create an HTML form with name, email, and message fields"_ — and let AI scaffold it for you, then customize as needed.
