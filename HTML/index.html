10. Advanced HTML

Details & Summary (Accordion)

What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML. It represents the page as a tree of objects that JavaScript can manipulate to change structure, style, and content dynamically.

What is a CSS Box Model?

Every HTML element is a rectangular box with: content, padding, border, and margin. Understanding the box model is fundamental to CSS layout.

What is Semantic HTML?

Semantic HTML uses elements with meaningful names that describe the role of their content — improving SEO, accessibility, and code readability.

Accessibility & ARIA

ARIA (Accessible Rich Internet Applications) adds accessibility metadata to elements for screen readers.

  • role — defines what the element is (e.g., role="dialog", role="alert", role="navigation")
  • aria-label — provides an accessible name for an element
  • aria-labelledby — points to another element that labels this one
  • aria-describedby — points to a description element
  • aria-hidden="true" — hides element from screen readers (decorative icons)
  • aria-live="polite" — announces dynamic content changes
  • aria-expanded="false" — indicates collapsed/expanded state
  • aria-required="true" — marks field as required
  • aria-invalid="true" — marks field as having an error
  • tabindex="0" — makes non-focusable elements keyboard-focusable
  • tabindex="-1" — removes element from tab order but allows focus via JS

Quick Reference Cheatsheet

Block-Level Elements

Always start on a new line and take full width:

div, p, h1–h6, ul, ol, li, table, form, fieldset, blockquote, pre, hr, header, main, footer, section, article, aside, nav, figure, details, dialog

Inline Elements

Flow within text, take only as much width as needed:

span, a, strong, em, b, i, u, s, del, ins, mark, code, kbd, abbr, cite, q, time, sub, sup, img, input, button, label, select, textarea, small, bdi, bdo, wbr

Void Elements (Self-Closing)

Cannot have children, no closing tag needed:

br, hr, img, input, link, meta, base, area, col, embed, param, source, track, wbr
