# Day 09 - Integrating OpenAI API

## Objective
Integrate an OpenAI-powered feature safely in a web app.

## Build
- frontend page
- backend/API proxy (recommended)
- `.env.example`

## Must Practice
- API key safety (never expose key in frontend)
- Request/response handling
- Error states and rate-limit handling

## Deliverable
A working AI assistant or text tool with loading/error UI states.

## Done Checklist
- [ ] API key stored in env variable
- [ ] User-friendly error handling
- [ ] Prompt + response logs for debugging
- [ ] Commit with `feat(day-09): openai integration`
